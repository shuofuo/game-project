## 一、游戏定位

```
名称：生肖天机
类型：放置合成游戏
平台：抖音小程序 / 快手小程序 / 微信小程序
目标用户：全年龄，华人 / 海外华人
是否需要版号：否（小游戏平台免版号）
```

### 核心玩法一句话概括

> 玩家选择自己的生肖（12选1）和命格（5选1），
> 通过召唤+合成，最终获得独一无二的"天命神兽"，
> 融入天干地支、二十四节气等中国传统命理文化。

## 📱 自适应设计规范（所有页面通用）

> 所有后续开发页面必须遵循以下规范，确保手机/平板/桌面全兼容

### 核心原则

```css
✅ 用 flex / grid 布局，不写死像素
✅ 容器：max-width + 90vw 双保险
✅ 字体：标题用 vw，业务内容用 rem
✅ 图片/视频：max-width: 100%
❌ 禁止写死 width: 960px / width: 1200px 等固定值
```

### 具体规范

| 项目 | 规则 |
|------|------|
| 主容器最大宽度 | `max-width: 500px; width: 90vw` |
| 最小适配宽度 | 320px（iPhone SE） |
| 网格列数 | 响应式：`repeat(auto-fit, minmax(80px, 1fr))` 或固定 4 列 |
| 间距 | 用 `rem` 或 `%`，不用 `px` 写死 |
| 字体大小 | 标题 `clamp(28px, 6vw, 52px)`，正文 `clamp(14px, 3vw, 18px)` |
| 按钮宽度 | `width: 100%` 或 `min(280px, 90vw)` |
| 背景 | 始终 `min-height: 100vh`，允许滚动 |

### 测试标准

每个页面完成后，在以下尺寸验证：
- 📱 手机：375×667 (iPhone SE) ✅
- 📱 大手机：390×844 (iPhone 14) ✅

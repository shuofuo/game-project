## 🎮 游戏简介

**水果合成家** - 拖拽两个相同水果合成更高阶水果的放置合成游戏。

```
合成链：
🥒 黄瓜 + 🥒 黄瓜 = 🍅 番茄
🍅 番茄 + 🍅 番茄 = 🍋 柠檬
🍋 柠檬 + 🍋 柠檬 = 🍊 橙子
🍊 橙子 + 🍊 橙子 = 🍎 苹果
🍎 苹果 + 🍎 苹果 = 🍐 梨
🍐 梨 + 🍐 梨 = 🍑 桃子
🍑 桃子 + 🍑 桃子 = 🍇 葡萄
🍇 葡萄 + 🍇 葡萄 = 🍉 西瓜
🍉 西瓜 + 🍉 西瓜 = 🌟 神奇水果（最高级）
```

**核心玩法**：
- 拖拽相同水果到合成区 → 合成更高阶水果
- 每种水果每秒产生金币
- 离线也能积累收益（最长8小时）
- 看激励视频加速收益 / 获得道具
- 金币用于升级产能、解锁新水果

## 🛠️ 技术栈

| 项目 | 技术选型 | 说明 |
|------|---------|------|
| 游戏引擎 | Cocos Creator 3.x | 免费跨平台，文档完善 |
| 开发语言 | TypeScript | Cocos官方推荐 |
| 代码管理 | GitHub | 代码同步，多设备协作 |
| AI编程辅助 | 硅基流动 API | ¥0.006/千token，每天¥3~15元 |
| 本地模型 | Ollama + Qwen2.5 | 零成本兜底 |
| 广告变现 | 穿山甲（字节）/ 快手广告联盟 | 按收入分成 |
| 数据存储 | 平台本地存储（PlayerPrefs）| 不需要服务器 |

**不需要：阿里云、自己的服务器、版号**

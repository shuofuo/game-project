# 🐾 生肖天机

> 一款融合中国传统命理文化的放置合成游戏  
> 目标平台：抖音小程序 / 快手小程序 / 微信小程序  
> 开发者：shuofuo + AI（基建带师 🛠️）

---

## 🎮 游戏简介

玩家选择自己的生肖（12选1）和命格（5选1），通过合成进化，最终获得独一无二的"天命神兽"。

```
🐀🐂🐅🐇🐉🐍🐎🐏🐒🐓🐕🐖  ×  🔥💧🪵⚪🟤 = 60种独特神兽
```

**核心玩法**：
- 召唤灵兽 → 拖拽合成更高等级
- 灵兽自动产金币 → 积累金币召唤更多
- 融入天干地支、二十四节气等传统文化
- 每日运势、天机降临、图鉴收集

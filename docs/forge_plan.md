# P1-2 炼宝阁增强

## 系统设计

### 核心概念
- **装备类型**: 武器(加金币CPS) / 护甲(加龙气) / 饰品(加属相碎片)
- **品质**: 普通→稀有→史诗→传说→天命 (白/蓝/紫/橙/红)
- **强化**: 消耗材料升级等级(1-15)，每次+属性
- **升星**: 消耗同名装备升星(1-5星)，升星后强化上限+3
- **套装**: 集齐武器+护甲+饰品 = 套装效果

### 材料系统
- 铁锭(iron): 普通，副本/挂机掉落
- 水晶(crystal): 稀有，完成成就掉落
- 龙鳞(dragonScale): 史诗，试炼塔掉落
- 星尘(starDust): 传说，品质召唤掉落

### 数据结构
```javascript
G.forge = {
  items: [
    {id:'w1', type:'weapon', name:'木剑', quality:0, star:1, level:1, equipped:false},
```

### 强化消耗
每级强化:
- 铁锭 += 10 × level
- 金币 += 50 × level²

### 升星消耗
| 当前星 | 需要同名装备 | 铁锭 |
|--------|------------|------|
| 1→2 | 2个1星 | 30 |
| 2→3 | 2个2星 | 80 |
| 3→4 | 2个3星 | 200 |
| 4→5 | 2个4星 | 500 |

### 套装效果
- 2件套: 全属相碎片掉率+20%
- 3件套: 金币产出+30% + 龙气产出+50%

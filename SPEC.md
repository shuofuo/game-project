## 一、游戏定位

```
名称：生肖天机
类型：放置合成游戏
平台：抖音小程序 / 快手小程序 / 微信小程序
目标用户：全年龄，华人 / 海外华人
是否需要版号：否（小游戏平台免版号）
```

### 核心玩法一句话概括

> 玩家选择自己的生肖（12选1）和命格（5选1），
> 通过合成进化，最终获得独一无二的"天命神兽"，
> 过程中融入天干地支、二十四节气等中国传统命理文化。

## 二、传统文化 → 游戏机制映射

| 现实文化 | 游戏机制 | 玩家感知 |
|---------|---------|---------|
| 十二生肖（属相）| 选生肖绑定，终身不变，决定最终神兽外形 | "这和我有关" |
| 天干地支（60甲子）| 实时时辰系统，每60分钟推进，特定组合触发天机事件 | "今天天机降临" |
| 五行（金木水火土）| 命格选择，5种不同属性加成 | "我选火命，产金快" |
| 二十四节气 / 农历节日| 限时活动，属相年庆典 | "今年是蛇年，产金×2" |
| 每日运势 | 随机1~5星，影响当日产出 | "今天运气爆棚！" |
| 图鉴收藏 | 解锁12属相图鉴 | "差一个就齐了" |

## 五、天机时辰系统

```
现实1小时 = 游戏内1个时辰
子丑寅卯辰巳午未申酉戌亥（12时辰/天）

每10个时辰（10小时）推进一个天干
天干：甲乙丙丁戊己庚辛壬癸（10个）
60天干地支组合，每60年（现实60天）完整循环一次

特殊天机事件：
- 🐉 龙抬头（农历二月初二）：全服产金×2
- ⚡ 天赦日（戊寅日）：召唤概率+50%
- 🔥 三伏天（未日午时）：火命产金×3
- 🌙 中秋（酉日）：龙气收益×3
- 🎉 甲子时：产金×5，全服狂欢
- 🐀 属相年：当年生肖产金全年×2
```

### Cocos Creator 项目结构

```
assets/
├── scenes/
│   ├── MainScene.fire          # 主场景（合成+产金+UI）
│   └── StartScene.fire         # 创建角色（选属相+命格）
├── scripts/
│   ├── GameManager.ts          # 游戏主循环
│   ├── MergeSystem.ts          # 合成逻辑（核心）
│   ├── CoinSystem.ts           # 金币系统
│   ├── DragonSystem.ts         # 灵兽管理
│   ├── QiSystem.ts             # 龙气系统
│   ├── TianJiSystem.ts         # 天机时辰系统
│   ├── FateSystem.ts           # 每日运势系统
│   ├── AdSystem.ts             # 广告接入
│   ├── SaveSystem.ts           # 本地存档
│   ├── ShareSystem.ts          # 分享系统
│   └── RankSystem.ts           # 排行榜（平台提供）
├── prefabs/
│   ├── creatures/              # 12等级灵兽预制体×12
│   ├── effects/                # 合成特效
│   └── ui/                     # 弹窗/面板预制体
├── textures/
│   ├── creatures/              # 灵兽图片
│   ├── ui/                     # 界面素材
│   └── effects/                # 粒子/光效
└── audio/
    ├── bgm/                    # 中国风背景乐
    └── sfx/                    # 合成音效/升级音效
```

## 二、成长阶段通用特征

> 每个灵兽的体型、细节、特效都随等级成长而升级，以下为通用规则。

### 2.1 体型变化（1→15级）

```
Lvl  1-3  ：圆润小巧（占格子60-70%宽高）  → 幼生感，圆滚滚
Lvl  4-6  ：身形展开（占格子70-80%）       → 开始有各生肖特征
Lvl  7-9  ：健美匀称（占格子80-90%）       → 特征鲜明，开始有灵气
Lvl 10-12 ：雄壮健美（占格子90-100%）      → 细节丰富，光效明显
Lvl 13-15 ：天命神化（占格子100-110%）     → 边框微微溢出，灵气环绕
```

### 2.2 通用视觉元素

| 元素 | 等级 1-3 | 等级 4-6 | 等级 7-9 | 等级 10-12 | 等级 13-15 |
|------|---------|---------|---------|----------|-----------|
| **灵气圈** | 无 | 1圈虚线 | 2圈虚线 | 3圈虚线 | 4圈+粒子光点 |
| **高光** | 单点高光 | 2点高光 | 渐变光 | 动态光 | 金色光晕+粒子 |
| **边框** | 单色细线 | 双色 | 双色+微光 | 三色+内发光 | 四色+外发光 |
| **特效** | 无 | 微微飘动 | 符文隐现 | 灵气蒸腾 | 神光天降 |
| **背景** | 单色圆 | 渐变圆 | 渐变+1圈 | 渐变+2-3圈 | 渐变+4圈+光点 |

### 2.3 眼睛设计（所有生肖通用）
- Lvl 1-6：黑瞳 + 白点高光（单纯可爱）
- Lvl 7-9：黑瞳 + 白点 + 彩色瞳孔边缘（灵动）
- Lvl 10-12：黑瞳 + 白点 + 金色瞳孔圈（威严）
- Lvl 13-15：黑瞳 + 白点 + 金色瞳孔圈 + 眼中光芒（神性）

### 4.2 皮肤叠加规则

1. **底色替换**：`skin.color` 作为卡片背景色、`skin.glow` 作为光晕色
2. **Filter叠加**：`skin.filter` 直接应用到 SVG `<img>` 元素（CSS filter）
3. **边框色**：`skin.border` 作为卡片边框和皮肤徽章底色
4. **阴影**：`skin.shadow` 作为 box-shadow 颜色
5. **皮肤专属特效**（SVG层叠）：
   - 金/天命：金色灵气圈 + 粒子光点 + L7+金色描边
   - 赤焰：火星粒子 + 火焰光效（L7+有真实火焰效果）
   - 玄青：青龙纹图案浮现 + 青色灵气（L10+有龙鳞光效）
   - 翡翠：玉质反光 + 绿色粒子
   - 紫霄：星辰粒子 + 紫色光芒
   - 白银：银色光环 + 冷色光效
   - 九幽：暗紫雾气 + 暗光粒子
   - 琉璃：彩虹色折射 + 透明质感
   - 天命：金色神光 + 多层光环

viewBox: "0 0 80 80"

### 5.3 升级为美术素材后的要求

**替换方案（仅改配置，不改代码）**：
1. 将美术图片放入 `docs/pet_imgs/` 目录
2. 命名格式：`{zodiac}_{level}.png`（或`.webp`）
3. 修改 `dragon_visual_config.json` 中对应记录的 `svgPath` 字段：
   ```json
   "svgPath": "docs/pet_imgs/4_15.webp"
   ```
4. `getDragonVisual()` 的 `_dragonIconHtml()` 会自动用 `<img>` 标签渲染

**图片规格建议**：

---

## 一、设计目标

玩家创建角色时选择 **12 生肖之一**，之后灵兽外观由三个维度叠加决定：

```
最终外观 ＝  生肖基础造型  ×  等级成长阶段  ×  皮肤材质特效
```

三者各自独立，每种组合（共 12 × 15 × 10 = 1800 种）都有独特视觉。

## 三、等级成长阶段（5 阶段 × 15 级）

| 阶段 | 等级 | 外观变化 |
|------|------|---------|
| **阶段1 幼生** | 1–3 级 | 基础幼兽形态，简单轮廓，只有一个基础生肖特征（耳/角/冠）。体型小，色彩浅，无光效。 |
| **阶段2 成长** | 4–6 级 | 体型明显变大，生肖特征强化（角更长、耳朵展开、身体比例更完整）。出现淡淡基础光晕（主体色 20% 透明度）。 |
| **阶段3 成形** | 7–9 级 | 身体比例趋向成熟，生肖特征完整（虎纹清晰、龙角成型、马鬃飘逸）。出现灵气光效（眼睛/角/背部有光点）。开始出现装饰元素（符文环/小披风/羽翼）。 |
| **阶段4 通灵** | 10–12 级 | 完全成熟形态，特征达到完整。出现武器感（爪套/角环/符文武器）和强烈灵气特效（环绕灵气、龙气缠绕）。高等级皮肤在此阶段出现环绕光点特效。 |
| **阶段5 神化** | 13–15 级 | **最终神兽形态**。体型最大，轮廓最复杂，光效最强。全身灵气包裹，特殊符文或光环。顶级皮肤出现拖尾效果。玩家看到"这就是我的守护神兽"。 |

### 视觉递增示例（以龙为例）

```
Lv1  简单小龙幼兽，无角，灰色身体
Lv4  龙角小芽出现，身体拉长，淡淡青光
Lv7  明显龙角，鳞片出现，背部青光
Lv10 龙角完整，鳞片清晰，灵气缠绕，爪套符文
Lv13 巨型神龙，全身金色灵气，神兽光环
```

## 五、大灵兽显示规则

大灵兽 = 生肖基础 SVG/Canvas 图形 + 等级阶段光效层 + 皮肤材质滤镜

```
大灵兽 HTML 结构（建议）：
<div class="dragon-display">
  <canvas class="dragon-body" />     ← 生肖基础造型（SVG/Canvas 绘制）
  <div class="dragon-stage-glow" />   ← 等级成长特效（光晕、灵气、符文）
  <div class="dragon-skin-overlay" /> ← 皮肤层（颜色滤镜、装饰、材质）
  <div class="dragon-skin-effect" />   ← 高级特效（Lv10+光点/拖尾/光环）
</div>
```

### 皮肤叠加优先级
1. 先绘制生肖基础轮廓（Canvas/SVG）
2. 再叠加等级成长特效（CSS animation + Canvas）
3. 最后应用皮肤颜色滤镜（CSS filter: sepia/hue-rotate/saturate）
4. Lv10+ 皮肤特效层叠加

## 八、技术实现建议

### 方案 A：纯 Canvas 绘制（推荐）
每个生肖画一个 Canvas path，level 决定 path 复杂度等级，skin 决定 color filter + 装饰叠加。

**优势**：体积小、灵活、可程序化控制；  
**劣势**：需要专业美术设计 path。

### 方案 B：CSS + SVG 组合
底色用 emoji/SVG 基础，生肖差异通过不同 SVG icon，等级通过 CSS scale + filter，皮肤通过 CSS hue-rotate + 额外装饰 PNG。

**优势**：实现快；  
**劣势**：生肖差异化有限（emoji 无法定制）。

### 方案 C：混合方案（推荐）
小格子和详情用 CSS/Emoji 过渡方案，正式版切换到 Canvas。

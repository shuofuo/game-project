## 一、当前布局问题诊断

### 问题汇总
| # | 问题 | 根因 |
|---|------|------|
| 1 | `grid-template-columns` 写 `52px 200px 1fr 52px`（4列）但 `grid-area` 用数字 1/2/3/4（4个独立编号）| Grid 4列但 grid-area 用的是绝对编号，不是行列定位 |
| 2 | 左侧竖排按钮 `grid-area:1`，右侧 `grid-area:4` | 同一行内两个元素用了相同行号，grid 渲染行为不确定 |
| 3 | `dragonGrid` 的 `grid-area:3` 但没有定义 grid 行列 | 整个 `middleRow` grid 未定义 `grid-template-rows`，列宽冲突 |
| 4 | `heroSection` 固定 200px 宽 | 小屏手机（如 375px）上只剩 175px 给格子，5×5根本放不下 |
| 5 | 无响应式断点 | 所有尺寸写死，手机端全面崩溃 |

## 二、新布局设计方案

### 核心思路：Flexbox 绝对主导

移除 Grid，改用 Flexbox，分层清晰：

```
┌────────────────────────────────────────────────┐
│  [🐉]  💰 12,345  +8/s  ✨ 520  [运势⭐⭐⭐]     │  ← HUD（刘海安全区）
├────────────────────────────────────────────────┤
│                                                │
│   [侧边栏]   灵兽格子（5×5 flex-grow）  [主角卡]│
│   📋签到                                        │
│   📋任务                                        │
│   ⚡活动                                        │
│   📖图鉴           格子区域（核心）    Lv.5     │
│   🏆排行                            🐉          │
│   🎖️成就                           灵兽详情    │
│   ⚙️设置                                        │
│                                                │
├────────────────────────────────────────────────┤
│            💰召唤  100   ✨召唤  500   🌿免费×3 │  ← 召唤栏
└────────────────────────────────────────────────┘
```

### 分区尺寸规范

```
┌─────────────────────────────────────────────────────┐
│  topHud       固定高度 ~52px（刘海安全区自动追加）    │
├──────────┬────────────────────────────┬────────────┤
│ 侧边栏    │    灵兽格子 5×5（核心区）    │  主角卡     │
│ 52px宽   │    flex:1（占满剩余空间）   │  140px宽   │
│ 固定     │    min-width:0 防止溢出     │  固定       │
├──────────┴────────────────────────────┴────────────┤
│  gameBottomBar  召唤栏（固定高度 ~64px）             │
└─────────────────────────────────────────────────────┘
```

### 各区详情

#### 1. topHud（顶部状态栏）
- 高度：`calc(env(safe-area-inset-top) + 40px)`
- 内容：属相图标 | 金币+产速 | 龙气 | 运势
- 背景：半透明毛玻璃 `rgba(7,7,26,.85)` + `backdrop-filter:blur(14px)`

#### 2. middleRow（主内容区）
- `display: flex; flex-direction: row;`
- **侧边栏**：52px 宽，`flex-shrink:0`，功能按钮竖排
- **灵兽格子**：`flex:1`，5×5 grid，`min-width:0`（防溢出）
- **主角卡**：140px 宽，`flex-shrink:0`，灵兽展示+等级+CPS

#### 3. skillBarRow（技能条）
- 固定在 topHud 下方或整合进 topHud（减少一行）
- 技能按钮横向滚动，`overflow-x:auto`

#### 4. gameBottomBar（底部召唤栏）
- 高度：`calc(64px + env(safe-area-inset-bottom,6px))`
- 召唤按钮居中，间隔 `gap: 12px`

### 响应式断点

| 设备 | 宽度 | 布局变化 |
|------|------|---------|
| 超小屏 | `<360px` | 主角卡缩小到 100px，格子自适应 |
| 小屏手机 | `360-480px` | 侧边栏缩到 44px，主角卡 120px |
| 普通手机 | `480-768px` | 标准布局 |
| 平板 | `>768px` | 主角卡扩展到 160px，侧边栏加宽到 60px |

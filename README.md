# 🐾 生肖天机

> 一款融合中国传统命理文化的放置合成游戏  
> 纯 HTML5 开发，浏览器即开即玩，GitHub Pages 零成本托管  
> 最终目标：抖音小程序 / 快手小程序 / 微信小程序

**🎮 在线体验：** https://shuofuo.github.io/game-project/  
**👤 开发者：** shuofuo + AI（基建带师 🛠️）

---

## 🎮 游戏简介

玩家选择自己的生肖（12选1），通过召唤灵兽、拖拽合成，获得 15 级灵兽，最终合成"鸿蒙神兽"。融入天干地支、五行命理等中国传统文化。

**核心循环：**
```
召唤 → 合成 → 产金币 → 再召唤 → 更强灵兽
```

## ✨ 已完成功能

### 召唤与合成
| 功能 | 说明 |
|------|------|
| 三种召唤方式 | 💰金币召唤 / ✨龙气召唤 / 🆓免费召唤 |
| 免费召唤倒计时 | 每日 00:00 重置，显示剩余次数 |
| 翻牌动画 | 召唤后翻转揭示灵兽品阶 |
| 拖拽合成 | 相同等级灵兽合并升一级（最高 Lv15 鸿蒙） |
| 合成闪光动画 | 合并时显示光效 |
| 合成音效 | 五行属性专属音效（12生肖各不同）|
| 召唤音效 | 翻牌 + 品阶分级和音 |
| 召唤结果通知 | 底部气泡，颜色随品阶变化 |

### 经济系统
| 功能 | 说明 |
|------|------|
| 自动产金币 | 每秒结算 CPS，金币飘字动画 |
| 金币产出加成 | 五行命格 × 每日运势 × 命格修炼 × 活动加成 |
| 龙气自然回复 | 命格修炼提升回复速度 |
| 合成随机龙气 | 有概率获得龙气 |

### 留存系统
| 功能 | 说明 |
|------|------|
| 每日签到 | 7天循环，断签重置，含金币/龙气/召唤次数奖励 |
| 每日任务 | 5个自动追踪任务（召唤/合成/登录），完成可领取奖励 |
| 限时活动 | 🎁周末双倍召唤 / 🌙晚间金币时段，叠加生效 |
| 每日运势 | 每日0点重置，1~5星影响金币产出 |

### 收集与成长
| 功能 | 说明 |
|------|------|
| 命格修炼 | 木/火/土/金/水，各3层，加成金币/召唤/合成/高级灵兽 |
| 灵兽图鉴 | 显示15级灵兽，灰色未解锁，新获得弹窗提示 |
| 成就系统 | 19个成就，居中大弹窗展示，达成音效 |
| 排行榜 | 段位系统（初窥→小成→大成→天师） |
| 18个成就徽章 | 召唤/合成/金币/收集四个维度 |

### 视觉与音效
| 功能 | 说明 |
|------|------|
| 灵兽详情弹窗 | 点击缩略图/主灵兽显示品阶+技能+产出 |
| 五行属相主题 | 12生肖各有主题色和背景动画 |
| 背景动态光效 | 随属相变化的星轨动画 |
| 侧边栏 | 签到/任务/活动/图鉴/排行榜/成就/设置 |
| 音效系统 | 召唤/合成/成就音效，背景音乐，五行专属BGM |
| 成就弹窗 | 居中大卡片，点击关闭，2.8秒后自动消失 |

### 数据持久化
| 功能 | 说明 |
|------|------|
| localStorage 自动存档 | 每10秒保存一次，召唤后立即保存 |
| 免费次数每日重置 | 00:00 自动刷新 |
| 签到/任务每日重置 | 同上 |
| 断签检测 | 连续天数精确管理 |

## 🛠️ 技术栈

| 项目 | 选型 | 说明 |

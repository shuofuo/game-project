# 生肖天机 - AI 灵兽图生成指南

## 一、风格规范

**所有图片必须统一以下风格：**

> **画风**：东方玄幻古风，赛博水墨风格（Ink + Cyberpunk Chinese Fantasy）
> **配色**：深紫/墨蓝背景 + 金/青/白渐变描边 + 粒子光效
> **构图**：圆形/圆角方形容器内，居中单角色，头像占 80%，留白 20%
> **质感**：金属光泽 + 灵气流动感 + 边缘光晕
> **输出**：512×512 PNG，透明背景或纯色背景，正方形
> **负面提示词**：写实照片、欧美卡通、3D渲染、低质量、模糊、水印
> **参考风格**：阴阳师式神卡牌 + 原神角色立绘 + 故宫文创插画

## 四、生成参数建议

| 参数 | 推荐值 |
|------|--------|
| 模型 | Midjourney v6 / DALL-E 3 / Stable Diffusion XL |
| 宽高比 | 1:1（正方形）|
| 风格 | `--style raw`（Midjourney）或 `natural`（DALL-E 3）|
| 质量 | `--q 1`（Midjourney）或 `hd`（DALL-E）|
| 变体 | 建议每个主题生成 4 张，选 1 张最好的 |

## 六、接入前端的方法

生成图片后，放到 `assets/` 目录，然后在 `config.js` 修改：

```javascript
// 生肖图（替换 ZOD_E emoji）
var ZOD_SPRITES = [
  null,  // 占位（未选）
  'assets/zodiac_01_rat.png',
  'assets/zodiac_02_ox.png',
  'assets/zodiac_03_tiger.png',
  'assets/zodiac_04_rabbit.png',
  'assets/zodiac_05_dragon.png',
  'assets/zodiac_06_snake.png',
  'assets/zodiac_07_horse.png',
  'assets/zodiac_08_goat.png',
  'assets/zodiac_09_monkey.png',
  'assets/zodiac_10_rooster.png',
  'assets/zodiac_11_dog.png',
  'assets/zodiac_12_pig.png',
];

// 灵兽图标（替换 LICON emoji）
var DRAGON_SPRITES = [
  null,
  'assets/dragon_lv01_egg.png',     // 灵蛋
  'assets/dragon_lv02_hatchling.png', // 幼灵
  ...（共15个）
];
```

然后修改 `renderGrid()` / `drawCell()` 等函数，用 `img.src = sprite` 替换 emoji 文字。

## 一、游戏定位

```
名称：生肖天机
类型：放置合成游戏
平台：抖音小程序 / 快手小程序 / 微信小程序
目标用户：全年龄，华人 / 海外华人
是否需要版号：否（小游戏平台免版号）
```

### 核心玩法一句话概括

> 玩家选择自己的生肖（12选1）和命格（5选1），
> 通过合成进化，最终获得独一无二的"天命神兽"，
> 过程中融入天干地支、二十四节气等中国传统命理文化。

## 二、传统文化 → 游戏机制映射

| 现实文化 | 游戏机制 | 玩家感知 |
|---------|---------|---------|
| 十二生肖（属相）| 选生肖绑定，终身不变，决定最终神兽外形 | "这和我有关" |
| 天干地支（60甲子）| 实时时辰系统，每60分钟推进，特定组合触发天机事件 | "今天天机降临" |
| 五行（金木水火土）| 命格选择，5种不同属性加成 | "我选火命，产金快" |
| 二十四节气 / 农历节日| 限时活动，今日属相加成 | "今天是我的属相日！产金+30%" |
| 每日运势 | 随机1~5星，影响当日产出 | "今天运气爆棚！" |
| 图鉴收藏 | 解锁12属相图鉴 | "差一个就齐了" |

## 五、天机时辰系统

```
现实1小时 = 游戏内1个时辰
子丑寅卯辰巳午未申酉戌亥（12时辰/天）

每10个时辰（10小时）推进一个天干
天干：甲乙丙丁戊己庚辛壬癸（10个）
60天干地支组合，每60年（现实60天）完整循环一次
```

### 5.3 天机事件体验设计

```
事件触发时的玩家体验：

① 前置提示（事件开始前5分钟）
   "天机预警：丙午时辰将至，届时产金×3..."

② 事件开始（全屏特效）
   - 屏幕中央："🔥 天机降临 · 丙午时辰"
   - 天降金色光柱，照在玩家的灵兽上
   - 背景音乐切换为气势磅礴的古风BGM

③ 事件进行中
   - 顶部横幅："天机进行中：产金×3，剩余4小时32分"
   - 灵兽发出金色光芒（视觉变化）

④ 事件结束
   - 倒计时结束，光芒散去
   - 弹窗："天机已过，下次降临..."

│   ├── SaveSystem.ts           # 本地存档
│   ├── ShareSystem.ts          # 分享系统
